## 사용스택 및 이유

- NextJS : Easy Routing & SSR
- chakra-ui : Simple & Clean UI
- prettier & eslint : Pretty Linting
- xterm : Fake Terminal UI
- React : UseEffect hook to Dynamic importing xterm in NextJS SSR
